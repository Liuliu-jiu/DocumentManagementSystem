# 项目介绍
这是一个基于qt的，用来管理文档的工具

# 项目描述
它是一个文档管理器，专门管理散落四处的文档，解决了文件四处散落，查找困难的问题，主要通过程序创建统一的库将文件收集并在库中集中管理，目前支持txt,word,excel,ppt文件，它不仅仅是一个管理器，也是一个知识整理中心

# 安装方法
通过点击安装包进行安装即可，卸载通过控制面板找到Document Management System version <版本号>卸载，如Document Management System version 1.0

# 使用方法
## (1)基本使用
### 1.新建笔记
    1.1 双击快捷方式打开程序
    1.2 文件 -> 新建笔记，选择您想要新建文件的后缀
    1.3 输入文件名
    1.4 此时新建的文件会出现在文档列表中

### 2.编辑笔记
    2.1 在文档列表找到想要编辑的笔记
    2.2 双击打开即可编辑

### 3.打开笔记(将系统文档加入至程序管理并编辑)
    3.1 文件 -> 打开笔记
    3.2 选择想要打开的软件名
    3.3 选择打开的文档
    3.4 此时文件会出现在文件列表中并会打开文件编辑
    
## (2)拓展功能
### 1.查找系统文件
    1.1 文件 -> 查找某个路径文件(txt,word,excel,ppt)
    1.2 选择你想要查找的目录路径
    1.3 搜索完成后会通过窗口的形式进行呈现

# 常见问题
## Q.支持windows和liunx系统吗
A.只支持windows系统

## Q.提示查找文档失败
A.查看路径是否在系统盘，如C:\Program Files，使用安装包安装时请将路径转移至其它盘

# 编译指南
## 环境
Qt版本：Qt6.8.3  
项目构建工具：Cmake(建议 >= 3.20)  
编译器：mingw  
版本控制工具：git
数据库：SQLite 3.51.1

## 步骤
### windows平台
#### 安装环境
1.推荐使用Qt官方在线安装器安装Qt creator  

#### 拉取代码
git@github.com:Liuliu-jiu/DocumentManagementSystem.git  

#### 项目构建
1.打开Qt Creator  
2.文件->打开文件或项目  
3.打开Cmake文件  
4.选择mingw编译器  
5.点击configure Project
